# Coffeeberry - coffee with a raspberry


## PyQt 5 & QML

 - buy coffe using RFID

# Physical devices

 - MFRC522 RFID Transmitter
 - HC-SR04 Ultrasonic Sensor
 - 16 Bit ADS for checking status LED
 - relays to switch start button
